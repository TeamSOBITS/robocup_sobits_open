# RoboCup SOBITS Open

RoboCup SOBITS Open Real Robot League(RRL) とSimulation Robot League(SRL)のタスク開発リポジトリになります．

ソースコード等は，各大会ごとのBranchに切り替えて，使用してください．
Branchの命名は，「大会の略称 + 開催年」にしてください．
